Giám sát trạng thái máy chủ#

Một số đề xuất#

Netdata : Giám sát thông số hệ thống

Mongoops : Backup, giám sát mongodb database

Ossec : Giám sát, theo dõi nguy cơ tấn công hệ thống, phản ứng ngăn chặn tự động

Jaeger : Giám sát thống kê traefik gateway

Giám sát trạng thái service#

docker service ls
docker stats
